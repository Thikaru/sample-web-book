# 🧪 Web Stack Lab (sample-web-stack)

HTML・CSS・JavaScriptの基礎を「説明 × 動く見本 × 書き換えられるエディタ」で学ぶ、育てていく見本集サイト。

ビルド不要。`index.html` をブラウザで開くだけで動きます。

## ページ構成

| ファイル | ページ | 内容 |
|---|---|---|
| `index.html` | トップ | サイト説明・本日の見本/一覧へのボタン・学習進捗 |
| `today.html` | 本日の見本 | 日替わりで見本を1つ表示(タイトル+説明+エディタ) |
| `samples.html` | 見本一覧 | カード表示 + カテゴリ/学習済みフィルタ |
| `sample.html?id=<見本id>` | 見本詳細 | タイトル+説明+ポイント+エディタ |
| `playground.html` | 遊び場 | 自由なエディタ。見本コードの取り込み・自動保存つき |

## ファイル構成

```
├── css/style.css       … 全ページ共通スタイル(CSS変数でテーマ管理)
├── js/data.js          … ★見本データ。ここに追記すると見本が増える
├── js/editor.js        … タブ切替エディタ + ライブプレビューの共通部品
└── js/sample-page.js   … today.html / sample.html の共通描画処理
```

## 見本の追加方法(定期更新はここだけ触ればOK)

`js/data.js` の `SAMPLES` 配列の末尾にオブジェクトを1つ追加するだけで、
一覧・詳細・本日の見本・遊び場のすべてに自動で反映されます。

```js
{
  id: "css-grid-layout",            // 一意な英数字ケバブケース(URLに使われる)
  title: "CSS Gridで格子レイアウト",
  category: "CSS",                  // "HTML" | "CSS" | "JS"
  level: "基礎",                    // "基礎" | "応用"
  modern: true,                     // ✨最新技術バッジ(省略可)。Baseline入り時期などを説明に書く
  date: "2026-07-08",               // 追加日 YYYY-MM-DD
  summary: "カード一覧に出す短い説明(1〜2文)。",
  description: "詳細ページに出す技術のしっかりした説明。",
  points: ["学びのポイント1", "ポイント2", "ポイント3"],
  html: `...初期HTMLコード...`,
  css: `...初期CSSコード...`,
  js: `...初期JSコード...`,
},
```

注意:
- コードはテンプレートリテラル(バッククォート)で書くため、コード内にバッククォートや `${` を含める場合は `\` でエスケープする。
- `date` を今日の日付にすると、その日は「本日の見本」に優先表示される。該当日がなければ日替わりローテーションになる。

## 見本以外の「育てる」ポイント

このサイト自体も学習の題材です。学んだ技術をサイト自身に適用していくと良い:

- `css/style.css` の `:root` のCSS変数を変えてテーマを模様替え
- トップページのヒーローアニメーションを学んだ技で強化
- エディタにシンタックスハイライトを追加(応用課題)
- 見本に「応用チャレンジ」欄を追加する、など

## 使っているデータ保存

- 学習進捗(「学んだ!」チェック): `localStorage` の `sws-learned`
- 遊び場のコード自動保存: `localStorage` の `sws-playground`
