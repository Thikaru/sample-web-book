このサイトでできること

📖

技術の説明を読む

それぞれの見本に「どんな技術か」「なにがポイントか」の解説つき。まず読んで、仕組みを理解できます。

👀

動く見本を見る

説明だけでは分からない「実際の動き」をその場で確認。見て、触って、体で覚えられます。

⌨️

コードを書き換える

HTML / CSS / JS のタブ付きエディタを搭載。数字や色を変えると、プレビューが即座に反応します。

🎡

遊び場で組み合わせる

学んだ見本のコードを取り込んで、自分だけの作品づくり。組み合わせてこそ、本当の理解に。
